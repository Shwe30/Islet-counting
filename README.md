# Islet-counting
This code identifies islets in a pancreas and counts the beta cells


This project performs:
- Islet segmentation using Otsu thresholding
- Nuclei detection using watershed
- Cell classification using K-means clustering

## Tools Used
- OpenCV
- Scikit-image
- NumPy
- Matplotlib

## Output
- Insulin-positive cell count
- Glucagon-positive cell count
- Islet area estimation
